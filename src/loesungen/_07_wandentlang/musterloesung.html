Wand entlang laufen

Kara läuft in einer Endlosschleife um die Bäume herum. Er betrachtet dazu in jedem Schritt die Sensoren "Baum vorne?" und "Baum rechts?".

while (true) {                                
  if (kara.treeFront() && kara.treeRight()) { 
    kara.turnLeft();                          
  }
  else if (!kara.treeFront())                 
    if (kara.treeRight()) {                   
      kara.move();                            
    }
    else {                                    
      kara.turnRight();                       
      kara.move();                            
    }
}

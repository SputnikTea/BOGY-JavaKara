PacMan – Kleeblätter fressen

Kara sucht in folgender Reihenfolge nach dem nächsten Kleeblatt (siehe untenstehende Abbildung):

  • Auf dem Feld vor ihm.
  • Auf dem Feld links von seiner Ausgangsposition.
  • Auf dem Feld rechts von seiner Ausgangsposition.

Folgender Programmteil führt diese Suche aus:

kara.move();      
if (!kara.onLeaf()) {     
  turnAround();           
  kara.move();
  kara.turnRight();
  kara.move();
  if (!kara.onLeaf()) {   
    turnAround();         
    kara.move();
    kara.move();
  }
}
kara.removeLeaf();
